# UCS761 - Deep Learning

This repository contains implementations of fundamental deep learning models developed and coursework completed as part of a UCS761 Deep Learning laboratory course.

## 📚 Course Overview
The course introduces the fundamental concepts and architectures of modern deep learning systems. The course covers Artificial Neural Networks, Convolutional Neural Networks, Recurrent Neural Networks, Autoencoders, and optimization techniques used for training deep learning models.

---

## Lab Exercises

### Lab 1 – Perceptron Logic Gates

Design and implementation of a perceptron model to simulate basic logic gates:

* AND
* OR
* NAND
* NOR

File:
`lab1-perceptron-logic-gates.ipynb`

---

### Lab 2 – Logistic Regression using Single Layer Perceptron

Implementation of logistic regression using a single layer perceptron (SLP) to classify types of glass from the Glass Identification dataset.

File:
`lab2-logistic-regression-slp-glass.ipynb`

Dataset:
`glass.csv`

---

### Lab 3 – Multiple Linear Regression using Perceptron

Implementation of multiple linear regression using a perceptron model trained with gradient descent.

File:
`lab3-multiple-linear-regression-perceptron.ipynb`

Dataset:
`multiple-linear-regression-dataset.csv`

---

### Lab 4 – Multilayer Perceptron with Backpropagation

Implementation of a multilayer perceptron (MLP) trained using backpropagation to predict the age of abalone based on physical measurements.

File:
`lab4-mlp-backpropagation-abalone.ipynb`

Dataset:
`abalone.data`
