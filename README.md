# UCS761 - Deep Learning

This repository contains assignments and coursework completed as part of the **UCS761 Deep Learning** course.

## 📚 Course Overview
The course introduces the fundamental concepts and architectures of modern deep learning systems. The course covers Artificial Neural Networks, Convolutional Neural Networks, Recurrent Neural Networks, Autoencoders, and optimization techniques used for training deep learning models.

## 📂 Repository Structure
Each assignment is organized in separate folders.

## 🎓 Purpose
This repository is maintained to track coursework progress and implementations related to the UCS761 Deep Learning course.
