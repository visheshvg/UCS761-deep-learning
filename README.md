# UCS761 - Deep Learning

This repository contains implementations of fundamental deep learning models and coursework completed as part of the **UCS761 Deep Learning** laboratory course.

---

## 📚 Course Overview

The course introduces the fundamental concepts and architectures of modern deep learning systems. It covers:

- Artificial Neural Networks
- Convolutional Neural Networks
- Recurrent Neural Networks
- Autoencoders
- Generative Adversarial Networks
- Optimization techniques used for training deep learning models

---

## 🧪 Lab Exercises

### Lab 1 – Perceptron Logic Gates
Design and implementation of a perceptron model to simulate basic logic gates:

- AND
- OR
- NAND
- NOR

**File:** `lab1-perceptron-logic-gates.ipynb`

---

### Lab 2 – Logistic Regression using Single Layer Perceptron
Implementation of logistic regression using a single layer perceptron (SLP) to classify types of glass from the Glass Identification dataset.

**File:** `lab2-logistic-regression-slp-glass.ipynb`  
**Dataset:** [Glass Identification Dataset](https://www.kaggle.com/datasets/uciml/glass)  
**Reference:** [Logistic Regression using Single Layer Perceptron Neural Network (SLPNN)](https://medium.com/analytics-vidhya/logistic-regression-using-single-layer-perceptron-neural-network-slpnn-31757c792d5d)

---

### Lab 3 – Multiple Linear Regression using Perceptron
Implementation of multiple linear regression using a perceptron model trained with gradient descent.

**File:** `lab3-multiple-linear-regression-perceptron.ipynb`  
**Dataset:** [Multiple Linear Regression Dataset](https://www.kaggle.com/datasets/hussainnasirkhan/multiple-linear-regression-dataset)

---

### Lab 4 – Multilayer Perceptron with Backpropagation
Implementation of a multilayer perceptron (MLP) trained using backpropagation to predict the age of abalone based on physical measurements.

**File:** `lab4-mlp-backpropagation-abalone.ipynb`  
**Dataset:** [Abalone Dataset](https://archive.ics.uci.edu/dataset/1/abalone)

## 🚀 Additional Assignments

### Lab 5 – Surface Defect Classification
Six kinds of typical surface defects of the hot-rolled steel strip are collected:

- Rolled-in scale (RS)
- Patches (Pa)
- Crazing (Cr)
- Pitted surface (PS)
- Inclusion (In)
- Scratches (Sc)

The database includes **1,800 grayscale images**, with **300 samples for each class**. Each class is split into **240 training images** and **60 testing images**.

**File:** `lab5-surface-defect-classification.ipynb`  
**Dataset:** [NEU Surface Defect Database](https://www.kaggle.com/datasets/kaustubhdikshit/neu-surface-defect-database)

**Models used:**
- ResNet50
- Google Inception Net
- EfficientNet

---

**Models used:**
